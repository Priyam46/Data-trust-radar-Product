# BI Integration Strategy

## Overview
Trust scores are exposed as a simple table consumable by BI tools.

## Integration Method
- Daily trust score table
- Joined with existing dataset metadata
- No plugins or custom visuals required

## Power BI
- Trust badge displayed using conditional formatting
- Tooltip explains score breakdown

## Tableau
- Calculated fields map trust score to color
- Hover text displays explanation

## Why this approach
- Zero friction adoption
- No BI admin dependency
- Scales across tools
